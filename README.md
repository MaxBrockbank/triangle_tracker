# Description

Tracks triangles.

## Required Technologies
* C#
* Text editor

## Set-up Instructions
1. Clone this repo to your desktop.
2. Open project directory in your text editor.
3. Navigator in your console to the `TriangleTracker` directory then run the command `dotnet run`.
